# symbol-complete

This is a very simple extension which converts a given set of patterns into a user-specified list of symbols.

## Extension Settings

```json
{
    "symbol-complete.replacements": {
        "\\|lambda\\|": "Î»"
    }
}
```

By default replacements for all Greek symbols are added.
